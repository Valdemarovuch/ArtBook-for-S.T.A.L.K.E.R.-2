# S.T.A.L.K.E.R. 2 — Електронний Артбук ☢️

Електронний трекер для колекціонування карток "Артбук S.T.A.L.K.E.R. 2" (з мережі супермаркетів АТБ). Створено фанатами для фанатів, щоб полегшити облік зібраних карток, пошук повторок та обмін з друзями.

![Stack](https://img.shields.io/badge/Stack-React%20+%20Vite%20+%20TailwindCSS-orange?style=flat-square)

## ✨ Особливості

- **Трекінг колекції:** Відмічай картки, які ти вже знайшов. Прогрес зберігається локально у твоєму браузері — жодних реєстрацій!
- **Облік повторок:** Додавай дублікати карток через контекстне меню (ПКМ на комп'ютері або **довге натискання** на мобільних пристроях).
- **Поділися з друзями:** Генеруй унікальне посилання зі своєю колекцією в один клік. Твої друзі зможуть побачити, яких карток тобі не вистачає, та які ти готовий віддати на обмін.
- **PDA-інтерфейс:** Стилізований темний дизайн, натхненний інтерфейсом КПК зі світу S.T.A.L.K.E.R. 2 з використанням плавних мікро-анімацій (Framer Motion).
- **Блискавична швидкість:** Всі зображення оптимізовані та переведені у сучасний формат WebP, що забезпечує миттєве завантаження навіть на мобільному інтернеті.
- **Адаптивність:** Ідеально працює як на комп'ютері, так і на мобільних пристроях.

Проєкт розгорнуто та доступний за адресою: **[artbook-stalker2.vercel.app](https://artbook-stalker2.vercel.app/)**

## 🚀 Як запустити локально

1. Встановіть залежності:
   ```bash
   npm install
   ```
2. Запустіть сервер для розробки:
   ```bash
   npm run dev
   ```
3. Відкрийте `http://localhost:5173` у вашому браузері.

Щоб зібрати проект для продакшену (наприклад, для завантаження на хостинг):
```bash
npm run build
```
